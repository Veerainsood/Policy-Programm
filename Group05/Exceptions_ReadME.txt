List of customr Exceptions implemented

1. Invalidchoice - This helps when the user selects an invalid choice(something which is not a choice). 
2. InvalidInputExeption - This helps when the data type entered does not match the expected data type.
3. InvalidPhNumExeption - This helps to reject invalid phone number entries by the user.
4. InvalidUserExeption - This rejects invalid user-id(which does not exist in database).
5. InvalidUserNameExeption - This prevents the user from having names which include special character or numbers.


Apart from this various standard exceptions are used across the code.
